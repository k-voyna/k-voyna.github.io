0.5b (2014.05.28)
-------------------------------------------------------------------------------
+ Добавлена возможность расчета по известной эффективно излучаемой мощности,
  наряду с расчетом по известной мощности передатчика.
- Для упрощения использования убраны модели реальных антенн. Теперь достаточно 
  выбрать одну из 5 моделей антенн. КПД антенн теперь не учитывается. 
  Учитывается диаграмма направленности и значение КНД для расчета по известной 
  мощности передатчика.
* Исправлено несколько досадных ошибок в расчетах эффективной излучаемой 
  мощности. Эмпирическая формула МККР для расчета напряженности поля 
  пространственной волны полностью совпала с формулой для расчета по 
  геометрической модели отражения от слоя E для его высоты в 100 км. 

0.4b (2014.05.26)
-------------------------------------------------------------------------------
* Скорректированы табличные параметры некоторых почв для приведения в 
  соответствие с использумыми в рекомендациях ITU-R.
+ Расчет глубины ближних замираний.
+ Расчет расстояния до конца зоны 50% ближних замираний.
+ Добавлена возможность ввода точности прогнозирования напряженности поля
  ионосферной волны (1%, 10%, 50%, 90%, 99%). 
* Повышена точность расчета напряженности поля земной волны. Коэффициенты 
  формулы Остина откалиброваны по графикам ITU-R.
+ Повышена точность расчета напряженности поля в ближней зоне. 
  Добавлена поправка по формулам ITU-R. 
+ Вывод в промежуточные результаты значений всех поправок.
* Произведена проверка формулы МККР и дополнительных поправок для 
  пространственной волны по физической модели, получено практически 
  полное соотвествие, в частности критерию сохранения энергии.
* Исправлена незначительная ошибка в определении начальной напряженности поля.

0.3b (2014.05.24)
-------------------------------------------------------------------------------
+ Ввод произвольных параметров почвы.
+ Поправки МККР для пространственной волны на время суток, магнитное наклонение
  и солнечную активность (для Европейского региона).
+ Учет затухания волн из-за кривизны поверхности Земли.
+ Возможность расчета на ДВ начиная от 150 кГц.
+ Две новых модели антенны: Т-образная укороченная для ДВ и эталонная 
  измерительная с КУ 1.5. 
* Реальные параметры антенн с учетом КПД.
